O **GameTracker** é um site criado para ajudar jogadores a **organizar e visualizar sua biblioteca de jogos** de forma simples e intuitiva.

O projeto apresenta uma interface moderna com navegação fácil e cards para exibição de jogos.

---

# 🚀 Funcionalidades

✔ Página inicial com apresentação do projeto  
✔ Carrossel de jogos em destaque  
✔ Página de cadastro de usuários  
✔ Biblioteca de jogos em formato de cards  
✔ Barra de navegação responsiva  
✔ Integração com **VLibras** para acessibilidade
